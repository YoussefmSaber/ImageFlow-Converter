# Changelog

All notable changes to **ImageFlow Converter** will be documented here.
Format follows [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2025-04-27

### Added
- WebP, PNG, JPG, conversion support
- Added a dedicated tool panel/tab
- Batch conversion support
- Image Scale support
- Image Quality Change support
- Generate density variant support
- Minimum requirement for the plugin in IntelliJ IDEA is 2022.1
- Minimum requirement for the plugin in Android studio is Android Studio Flamingo (2022.2)
